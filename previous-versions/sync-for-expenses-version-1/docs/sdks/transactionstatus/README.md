# TransactionStatus
(*transactionStatus*)

## Overview

Retrieve the status of transactions within a sync.

### Available Operations

* [getSyncTransaction](#getsynctransaction) - Get sync transaction
* [listSyncTransactions](#listsynctransactions) - Get sync transactions
