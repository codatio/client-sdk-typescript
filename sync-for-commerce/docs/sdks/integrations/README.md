# Integrations
(*integrations*)

## Overview

View useful information about codat's integrations.

### Available Operations

* [getBranding](#getbranding) - Get branding for an integration
* [list](#list) - List integrations

## getBranding

Retrieve Integration branding assets.

### Example Usage

```typescript
import { CodatSyncCommerce } from "@codat/sync-for-commerce";

async function run() {
  const sdk = new CodatSyncCommerce({
    security: {
      authHeader: "Basic BASE_64_ENCODED(API_KEY)",
    },
  });

  const result = await sdk.integrations.getBranding({
    platformKey: "gbol",
  });

  // Handle the result
  console.log(result)
}

run();
```

## list

Retrieve a list of available integrations support by data type and state of release.

### Example Usage

```typescript
import { CodatSyncCommerce } from "@codat/sync-for-commerce";

async function run() {
  const sdk = new CodatSyncCommerce({
    security: {
      authHeader: "Basic BASE_64_ENCODED(API_KEY)",
    },
  });

  const result = await sdk.integrations.list({
    orderBy: "-modifiedDate",
    page: 1,
    pageSize: 100,
  });

  // Handle the result
  console.log(result)
}

run();
```
