## getBranding

### Example Usage

```typescript

  const sdk = new CodatSyncCommerce({
    security: {
      authHeader: "Basic BASE_64_ENCODED(API_KEY)",
    },
  });

  const res = await sdk.integrations.getBranding({
    platformKey: "gbol",

    // handle response

})();
```
