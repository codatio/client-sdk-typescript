# AdvancedControls

## Overview

### Available Operations

* [createCompany](#createcompany) - Create company
* [getConfiguration](#getconfiguration) - Get company configuration
* [listCompanies](#listcompanies) - List companies
* [setConfiguration](#setconfiguration) - Set configuration

## createCompany

Creates a Codat company

### Example Usage

```typescript
import { CodatSyncCommerce } from "@codat/sync-for-commerce";

async function run() {
  const sdk = new CodatSyncCommerce({
    security: {
      authHeader: "Basic BASE_64_ENCODED(API_KEY)",
    },
  });

  const result = await sdk.advancedControls.createCompany({
    description: "Requested early access to the new financing scheme.",
    groups: [
      {
        id: "60d2fa12-8a04-11ee-b9d1-0242ac120002",
      },
    ],
    name: "Bank of Dave",
  });

  // Handle the result
  console.log(result)
}

run();
```
