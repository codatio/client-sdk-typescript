# IntegrationsSDK
(*integrations*)

## Overview

Get a list of integrations supported by Codat and their logos.

### Available Operations

* [get](#get) - Get integration
* [getBranding](#getbranding) - Get branding
* [list](#list) - List integrations

## get

Get single integration, by platformKey

### Example Usage

```typescript
import { CodatPlatform } from "@codat/platform";

const codatPlatform = new CodatPlatform({
  authHeader: "Basic BASE_64_ENCODED(API_KEY)",
});

async function run() {
  const result = await codatPlatform.integrations.get({
    platformKey: "gbol",
  });

  // Handle the result
  console.log(result)
}

run();
```

## list

List your available integrations

### Example Usage

```typescript

const codatPlatform = new CodatPlatform({
  authHeader: "Basic BASE_64_ENCODED(API_KEY)",
});

async function run() {
  const result = await codatPlatform.integrations.list({
    orderBy: "-modifiedDate",
    page: 1,
    pageSize: 100,
    query: "id=e3334455-1aed-4e71-ab43-6bccf12092ee",
  });

  // Handle the result
  console.log(result)
}

run();
```
