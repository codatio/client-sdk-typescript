# TransferAccount

Account details of the account sending or receiving the transfer.

## Example Usage

```typescript
import { TransferAccount } from "@codat/lending/sdk/models/shared";

let value: TransferAccount = {
  accountRef: {
    dataType: "accountTransaction",
  },
  currency: "GBP",
};
```
